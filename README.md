# AI Image Classification Web App

**Laravel 12.5 + Flask + PyTorch**

This project is a web-based AI application where users upload an image and receive a prediction generated by a PyTorch deep learning model.
The system integrates a Laravel frontend with a Python Flask API that loads a trained `.pt` model and performs inference.

# рџ“Ў API Flow Example

1. User uploads an image
2. Laravel stores the image
3. Laravel sends the image to Flask
4. Flask loads the `.pt` model
5. Prediction generated
6. JSON returned to Laravel
7. Laravel saves result in database
8. Result displayed to user
